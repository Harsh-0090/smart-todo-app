# Smart To-Do App

A simple and interactive **Smart To-Do App** built with **React.js** to manage tasks efficiently.  
Users can **add, search, and filter tasks** with a clean and responsive UI.

## Features
- Add tasks using **Input Box** and **Add Button**
- **Search tasks** by keywords
- **Filter tasks**: All / Completed
- **Responsive and clean UI** for better user experience

## Technologies Used
- React.js
- HTML, CSS, JavaScript

## How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Harsh-0090/smart-todo-app.git

2. Navigate to the project folder:
   ```bash
   cd smart-todo-app

3. Install dependencies:
   ```bash
   npm install
   
4. Start the application:
   ```bash
   npm start

5. Open in browser:

   http://localhost:3000

   
  ### 🔹 Explanation:
- **Step 1 → git clone** : GitHub se project copy karne ke liye  
- **Step 2 → cd smart-todo-app** : Folder ke andar jao  
- **Step 3 → npm install** : Required dependencies install karo  
- **Step 4 → npm start** : React app run karo  
- **Step 5 → browser link** : Localhost me app open karo
